Detailed Description

Armazena todos os atributos de um determinado usuário.

A classe usuario é responsável por armazenar os atributos necessários para os usuários, ao qual serão necessárias para inicializar o sistema, de acordo com um usuário específico, esta função dá a possibilidade de alterar e retornar quaisquer atributos, ela possui funções virtuais, aos quais serão utilizadas pelas filhas
